ARTIST STATEMENT // PURPOSE:

I have leaned lots about web development through this program and would love to see how far I take it, the website you are currently viewing has been coded by me from scratch using Visual Studio code. I feel it is necessary for me to be in the Digital Media Arts program as many of my aspirations align with the content that is taught. I wish to continue learning new skills that will broaden my potential of what I can create to further my existing goals.
