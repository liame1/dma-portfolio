JANUARY POSTER - ART 74

This is a poster I made using photoshop, the letters in the top-left section were done by a good friend of mine, which I then scanned and edited to fit. The cityscape was created by using a screenshot I took from the video game Jet Set Radio Future.

NINTENDO SWITCH VECTOR - ART 74

This was a image I created using the pen tool in Illustrator. I used a photo of my own nintendo switch for reference when creating the vectors, and used photoshop to add the finishing details. The goal of this project was to create an image that would resemble a product advertisment.

LASERCUT RACECAR - ART 13

This was a model car I laser-cut using pen vecotrs I created in Illustrator. I had no prior expierience laser-cutting anything before, but it was very fun learn the process. When designing the car, I imagined each piece and how they would be layered together. I used the design of formula-1 racecars as reference, but wanted to simplified the cars details as much as possible while also retaining its key identifiable features.
